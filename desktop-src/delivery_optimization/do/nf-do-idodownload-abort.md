# IDODownload::Abort method

Aborts the download.

## Syntax

```cpp
HRESULT Abort();
```

## Return Value

If the function succeeds, it returns **S_OK**. Otherwise, it returns an [**HRESULT**](/windows/desktop/com/structure-of-com-error-codes) [error code](/windows/desktop/com/com-error-codes-10).
