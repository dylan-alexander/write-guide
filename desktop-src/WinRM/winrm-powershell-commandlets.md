# Managed Reference for WinRM Windows PowerShell Command Classes

Windows Remote Management 2.0 (WinRM) can use Windows PowerShell cmdlets for system management. Windows PowerShell cmdlets enable an administrator to configure WinRM and to get data or manage resources.

Windows PowerShell cmdlets for WinRM provide the same functionality as the winrm command line utility. However, Windows PowerShell also does the following:

-   Automates WinRM tasks in an extensible and management-oriented scripting language.
-   Provides a single tool for all management tasks.

See [Windows PowerShell](https://go.microsoft.com/fwlink/p/?linkid=142509) for more information.

## WS-Management Windows PowerShell Classes

**Namespace**: Microsoft.WsMan.Management

**Assembly**: System.Management.Automation

The following WinRM command classes are implemented by Windows PowerShell. These classes are included in this software development kit (SDK) for completeness only. The members of these classes cannot be used directly nor should they be used to derive other classes.



| Class                        | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
|------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ConnectWSManCommand          | Connects to the WinRM service on a remote computer. <br/> See the [Connect-WSMan](https://go.microsoft.com/fwlink/p/?linkid=155382) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                              |
| DisableWSManCredSSPCommand   | Disables CredSSP authentication on a client.<br/> See the [Disable-WSManCredSSP](https://go.microsoft.com/fwlink/p/?linkid=155383) cmdlet for detailed information about the parameters and for examples.<br/>                                                                                                                                                                                                                                                                                                                                           |
| DisconnectWSManCommand       | Disconnects from the WinRM service on a remote computer. <br/> See the [Disconnect-WSMan](https://go.microsoft.com/fwlink/p/?linkid=155384) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                      |
| EnableWSManCredSSPCommand    | Enables CredSSP authentication on the client.<br/> See the [Enable-WSManCredSSP](https://go.microsoft.com/fwlink/p/?linkid=155385) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                               |
| GetWSManCredSSPCommand       | Gets the CredSSP-related configuration for the client.<br/> See the [Get-WSManCredSSP](https://go.microsoft.com/fwlink/p/?linkid=155386) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                         |
| GetWSManInstanceCommand      | Displays management information for a resource instance specified by a Resource URI.<br/> See the [Get-WSManInstance](https://go.microsoft.com/fwlink/p/?linkid=155387) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                          |
| InvokeWSManActionCommand     | Invokes an action on the target object specified by the Resource URI and selectors. <br/> See the [Invoke-WSManAction](https://go.microsoft.com/fwlink/p/?linkid=155388) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                         |
| NewWSManInstanceCommand      | Creates a new instance of a management resource. <br/> See the [New-WSManInstance](https://go.microsoft.com/fwlink/p/?linkid=155389) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                             |
| NewWSManSessionOptionCommand | Creates a WinRM Session option hash table to use as input parameters for the following WSMan cmdlets: [Get-WSManInstance](https://go.microsoft.com/fwlink/p/?linkid=155387), [Set-WSManInstance](https://go.microsoft.com/fwlink/p/?linkid=155390), [Invoke-WSManAction](https://go.microsoft.com/fwlink/p/?linkid=155388), and [Connect-WSMan](https://go.microsoft.com/fwlink/p/?linkid=155382). <br/> See the [New-WSManSessionOption](https://go.microsoft.com/fwlink/p/?linkid=155391) for examples and detailed information about the parameters.<br/> |
| RemoveWSManInstanceCommand   | Deletes a management resource instance. <br/> See the [Remove-WSManInstance](https://go.microsoft.com/fwlink/p/?linkid=155392) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| SetWSManInstanceCommand      | Modifies management information related to a resource. <br/> See the [Set-WSManInstance](https://go.microsoft.com/fwlink/p/?linkid=155390) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                       |
| SetWSManQuickConfigCommand   | Configures the local computer for remote management. <br/> See the [Set-WSManQuickConfig](https://go.microsoft.com/fwlink/p/?linkid=155393) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                                      |
| TestWSManCommand             | Tests whether the WinRM service is running on a local or remote computer. <br/> See the [Test-WSMan]( https://go.microsoft.com/fwlink/p/?linkid=155394) cmdlet for examples and detailed information about the parameters.<br/>                                                                                                                                                                                                                                                                                                                        |
| SessionOption                | Defines a set of extended options for the WS-Management session.<br/> See the [Connect-WSMan](https://go.microsoft.com/fwlink/p/?linkid=155382) cmdlet for examples and detailed information about the possible values.<br/>                                                                                                                                                                                                                                                                                                                             |
