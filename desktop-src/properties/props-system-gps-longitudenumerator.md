# System.GPS.LongitudeNumerator

The numerator of PKEY\_GPS\_Longitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.LongitudeNumerator
   shellPKey = PKEY_GPS_LongitudeNumerator
   formatID = 02B0F689-A914-4E45-821D-1DDA452ED2C4
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt32
```

## Remarks

PKEY values are defined in Propkey.h.
