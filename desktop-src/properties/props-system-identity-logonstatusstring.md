# System.Identity.LogonStatusString

A string that indicates the user logon status of an identity.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.LogonStatusString
   shellPKey = PKEY_Identity_LogonStatusString
   formatID = F18DEDF3-337F-42C0-9E03-CEE08708A8C3
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
