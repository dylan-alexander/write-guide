# System.Media.SeasonNumber

A 1 based monotonically incremented number that corresponds to the season that the show was first presented

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Media.SeasonNumber
   shellPKey = PKEY_Media_SeasonNumber
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## Remarks

PKEY values are defined in Propkey.h.
