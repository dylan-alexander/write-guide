# System.PriorityText

The user-friendly form of [System.Priority](https://msdn.microsoft.com/library/Bb787550(v=VS.85).aspx). This value is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PriorityText
   shellPKey = PKEY_PriorityText
   formatID = D98BE98B-B86B-4095-BF52-9D23B2E0A752
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
