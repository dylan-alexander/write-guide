# System.DataObjectFormat

The data object format. A string value that is the clipboard format name.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.DataObjectFormat
   shellPKey = PKEY_DataObjectFormat
   formatID = 1E81A3F8-A30F-4247-B9EE-1D0368A9425C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## Remarks

PKEY values are defined in Propkey.h.
