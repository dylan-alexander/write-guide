# System.Devices.InLocalMachineContainer

Device is in the local machine container.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.InLocalMachineContainer
   shellPKey = PKEY_Devices_InLocalMachineContainer
   formatID = 8C7ED206-3F8A-4827-B3AB-AE9E1FAEFC6C
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
