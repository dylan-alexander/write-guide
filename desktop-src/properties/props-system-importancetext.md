# System.ImportanceText

The user-friendly form of [System.Importance](https://msdn.microsoft.com/library/Bb760723(v=VS.85).aspx). This value is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ImportanceText
   shellPKey = PKEY_ImportanceText
   formatID = A3B29791-7713-4E1D-BB40-17DB85F01831
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
