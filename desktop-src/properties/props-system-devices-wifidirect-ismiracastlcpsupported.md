# System.Devices.WiFiDirect.IsMiracastLcpSupported

Indicates if link content protection is supported by the Wi-Fi Direct Device if it is Miracast capable

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsMiracastLcpSupported
   shellPKey = PKEY_Devices_WiFiDirect_IsMiracastLcpSupported
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
