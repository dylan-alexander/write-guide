# System.GPS.DestBearingRef

Indicates the reference used for giving the bearing to the destination point (for example,true direction, magnetic direction).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestBearingRef
   shellPKey = PKEY_GPS_DestBearingRef
   formatID = 9AB84393-2A0F-4B75-BB22-7279786977CB
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
