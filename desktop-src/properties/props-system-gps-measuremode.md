# System.GPS.MeasureMode

Indicates the GPS measurement mode (for example, two-dimensional, three-dimensional).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.MeasureMode
   shellPKey = PKEY_GPS_MeasureMode
   formatID = A015ED5D-AAEA-4D58-8A86-3C586920EA0B
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
