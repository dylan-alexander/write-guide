# System.Devices.GlyphIcon

Glyph Icon Path.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.GlyphIcon
   shellPKey = PKEY_Devices_GlyphIcon
   formatID = 51236583-0C4A-4FE8-B81F-166AEC13F510
   propID = 123
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

PKEY values are defined in Propkey.h.
