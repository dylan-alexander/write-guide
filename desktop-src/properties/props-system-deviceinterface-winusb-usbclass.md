# System.DeviceInterface.WinUsb.UsbClass

Class value from the USB device's first USB Interface Descriptor.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbClass
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbClass
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
```
