# System.Devices.Aep.Category

Categories the device is part of. e.g. Printer, Camera, etc.

```
propertyDescription
   name = System.Devices.Aep.Category
   shellPKey = PKEY_Devices_Aep_Category
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
```
