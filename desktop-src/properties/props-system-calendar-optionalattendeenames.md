# System.Calendar.OptionalAttendeeNames

Names of the all the optional attendees.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.OptionalAttendeeNames
   shellPKey = PKEY_Calendar_OptionalAttendeeNames
   formatID = 09429607-582D-437F-84C3-DE93A2B24C3C
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

PKEY values are defined in Propkey.h.
