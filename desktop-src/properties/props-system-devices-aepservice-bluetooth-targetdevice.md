# System.Devices.AepService.Bluetooth.TargetDevice

Bluetooth parent device for the query. Required for uncached queries.

## Windows 10, version 1703, Windows 10, version 1607

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.TargetDevice
   shellPKey = PKEY_Devices_AepService_Bluetooth_TargetDevice
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
```
