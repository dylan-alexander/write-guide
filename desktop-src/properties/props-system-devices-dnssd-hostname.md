# System.Devices.Dnssd.HostName

DNS name of device is hosting the service.

```

   name = System.Devices.Dnssd.HostName
   shellPKey = PKEY_Devices_Dnssd_HostName

   propID = 7
   SearchInfo
      InInvertedIndex = false
```
