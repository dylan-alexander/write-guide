# System.Audio.IsVariableBitRate

Indicates whether the audio file had a variable or constant bit rate.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Audio.IsVariableBitRate
   shellPKey = PKEY_Audio_IsVariableBitRate
   formatID = E6822FEE-8C17-4D62-823C-8E9CFCBD1D5C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
