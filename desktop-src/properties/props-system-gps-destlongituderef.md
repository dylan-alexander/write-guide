# System.GPS.DestLongitudeRef

Indicates whether the longitude destination point is east or west longitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestLongitudeRef
   shellPKey = PKEY_GPS_DestLongitudeRef
   formatID = 182C1EA6-7C1C-4083-AB4B-AC6C9F4ED128
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
