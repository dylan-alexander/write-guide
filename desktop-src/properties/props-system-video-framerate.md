# System.Video.FrameRate

Indicates the frame rate for the video stream, in frames per 1000 seconds.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Video.FrameRate
   shellPKey = PKEY_Video_FrameRate
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.Video.FrameRate
   shellPKey = PKEY_Video_FrameRate
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 6
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

To help reduce truncation error, this property does not use the standard frame rate measure of frames per second (FPS). Instead, this property measures frame rate as frames per 1000 seconds (FPS multiplied by 1000). For example, [System.Video.FrameRate](https://msdn.microsoft.com/library/Bb760066(v=VS.85).aspx) would express a frame rate of 29.97 FPS as an integer value of 29970.

PKEY values are defined in Propkey.h.
