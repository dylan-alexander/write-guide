# System.Volume.FileSystem

The name of a volume's file system. This is valid for Shell items that describe a volume.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Volume.FileSystem
   shellPKey = PKEY_Volume_FileSystem
   formatID = 9B174B35-40FF-11D2-A27E-00C04FC30871
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
