# System.Search.GatherTime

The Datetime value representing the time when the Windows Search Gatherer process last pushed properties of this document to the Windows Search Gatherer Plugins.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Search.GatherTime
   shellPKey = PKEY_Search_GatherTime
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.Search.GatherTime
   shellPKey = PKEY_Search_GatherTime
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 8
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
