# System.DeviceInterface.Serial.UsbProductId

Serial device USB Product Id

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.DeviceInterface.Serial.UsbProductId
   shellPKey = PKEY_DeviceInterface_Serial_UsbProductId
   formatID = 4C6BF15C-4C03-4AAC-91F5-64C0F852BCF4
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

PKEY values are defined in Propkey.h.
