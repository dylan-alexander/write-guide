# System.Calendar.OrganizerAddress

The address of the event organizer. This is a mailing or street address.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.OrganizerAddress
   shellPKey = PKEY_Calendar_OrganizerAddress
   formatID = 744C8242-4DF5-456C-AB9E-014EFB9021E3
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
