# System.Photo.ContrastText

The user-friendly form of System.Photo.Contrast. It is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ContrastText
   shellPKey = PKEY_Photo_ContrastText
   formatID = 59DDE9F2-5253-40EA-9A8B-479E96C6249A
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true

      type = String
```
