# System.FileAttributes

The attributes of the item. These are equivalent to the values recognized in the **dwFileAttributes** member of the [**WIN32\_FIND\_DATA**](https://msdn.microsoft.com/en-us/library/Aa365740(v=VS.85).aspx) structure.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.FileAttributes
   shellPKey = PKEY_FileAttributes
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
