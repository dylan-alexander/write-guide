# Load(int) function

Gets one value.

## Syntax


```C++
uint Load(
  _In_ int Location
);
```



## Parameters

<dl> <dt>

*Location* \[in\]
</dt> <dd>

Type: **int**

The location of the buffer.

</dd> </dl>

## Return value

Type: **uint**

One value.
