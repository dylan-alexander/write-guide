# QoS Templates in the SPI

The Quality of Service (QoS) implementation details are provided in a separate, dedicated QoS section in the Microsoft Windows Software Development Kit (SDK). Please see the [Quality of Service](https://msdn.microsoft.com/en-US/library/Aa374094(v=VS.80).aspx) section of the Windows SDK for information on QoS usage guidelines, including available QoS templates and their associated [**FLOWSPEC**](https://msdn.microsoft.com/en-US/library/Aa373702(v=VS.80).aspx) values.
