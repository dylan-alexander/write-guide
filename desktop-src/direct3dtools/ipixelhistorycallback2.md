# <span id="vspixengine.ipixelhistorycallback2"></span>IPixelHistoryCallback2 interface

Callback to return pixel history intersections (draw call level) and primitives (triangle level) in two different results.

## Members

The **IPixelHistoryCallback2** interface inherits from the [**IUnknown**](https://docs.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface. **IPixelHistoryCallback2** also has these types of members:
