# Using the Certificate Services Backup and Restore Functions

Certificate Services provides a set of functions which can be used to back up and restore the certificate services database files. The following sections describe how these functions may be used.



| Section                                                                                                                            | Information                                                                                                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Setting the Backup and Restore Privileges](setting-the-backup-and-restore-privileges.md)                                         | Setting the necessary [*privileges*](https://msdn.microsoft.com/en-us/library/ms721603(v=VS.85).aspx) to call the Certificate Services backup and restore functions                      |
| [Backing Up Certificate Services](backing-up-certificate-services.md)                                                             | Backup scenario using the backup functions                                                                                                                                              |
| [Restoring Certificate Services from Backup](restoring-certificate-services-from-backup.md)                                       | Restore scenario using the backup functions                                                                                                                                             |
| [Backing Up and Restoring the Certificate Services Private Key](backing-up-and-restoring-the-certificate-services-private-key.md) | Backing up and restoring [*private keys*](https://msdn.microsoft.com/en-us/library/ms721603(v=VS.85).aspx), which cannot be backed up or restored with the Certificate Services API. |
