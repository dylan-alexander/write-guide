# PROV\_RSA\_SIG

The PROV\_RSA\_SIG provider type is a subset of PROV\_RSA\_FULL. It supports only those functions and algorithms required for [*hashes*](https://msdn.microsoft.com/en-us/library/ms721586(v=VS.85).aspx) and [*digital signatures*](https://msdn.microsoft.com/en-us/library/ms721573(v=VS.85).aspx).

## Algorithms Supported

For descriptions of each of these algorithms, see the glossary.



| Purpose      | Supported algorithms                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Key Exchange | None                                                                                                                              |
| Signature    | [*RSA*](https://msdn.microsoft.com/en-us/library/ms721604(v=VS.85).aspx)                                                                       |
| Encryption   | None                                                                                                                              |
| Hashing      | [*MD5*](https://msdn.microsoft.com/en-us/library/ms721594(v=VS.85).aspx)[*SHA*](https://msdn.microsoft.com/en-us/library/ms721625(v=VS.85).aspx)<br/> |



 

## Related Documentation

The documentation for the [PROV\_RSA\_FULL](prov-rsa-full.md) provider type also applies to the PROV\_RSA\_SIG provider type.
