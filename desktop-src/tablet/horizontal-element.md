# Horizontal Element

Contains formatting information for the horizontal lines used for the stationery in a Journal note.

## Definition

``` syntax
<xs:element name="Horizontal" type="HorizontalType" minOccurs="0" />
```

## Parent Elements

## Child Elements

None.
