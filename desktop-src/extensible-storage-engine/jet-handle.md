# JET_HANDLE


_**Applies to:** Windows | Windows Server_

## JET_HANDLE

The **JET_HANDLE** data type contains a generic handle.

```cpp
    typedef JET_API_PTR JET_HANDLE;
```

### Data Types

JET_HANDLE

A value of **NULL** indicates an invalid handle.
