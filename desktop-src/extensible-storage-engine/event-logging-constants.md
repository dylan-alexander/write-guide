# Event Logging Constants


_**Applies to:** Windows | Windows Server_

## Event Logging Constants

The following constants indicate the detail level for event log messages for the [JET_ParamEventLoggingLevel](gg294086\(v=exchg.10\).md) system parameter.

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><p>Constant/value</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>JET_EventLoggingDisable<br />
0</p></td>
<td><p>Disables event logging.</p></td>
</tr>
<tr class="even">
<td><p>JET_EventLoggingLevelMax<br />
100</p></td>
<td><p>Sets the maximum level to use for event logs, which is currently 100 characters.</p></td>
</tr>
</tbody>
</table>
