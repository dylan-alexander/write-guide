# Error Handling Constants


_**Applies to:** Windows | Windows Server_

## Error Handling Constants

The following constants are used to set the range for the [JET_paramExceptionAction](gg269173\(v=exchg.10\).md) system parameters.

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><p>Constant/value</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>JET_ExceptionMsgBox<br />
0x0001</p></td>
<td><p>Displays a message box when an exception occurs.</p></td>
</tr>
<tr class="even">
<td><p>JET_ExceptionNone<br />
0x0002</p></td>
<td><p>Does nothing when an exception occurs.</p></td>
</tr>
</tbody>
</table>
