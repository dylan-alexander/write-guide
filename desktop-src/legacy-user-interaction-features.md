# Legacy User Interaction Features

User interaction features for Windows 7 and earlier.

## In this section



| Topic                                                               | Description                                                                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Touch Input](https://msdn.microsoft.com/en-us/library/Dd562197(v=VS.85).aspx)<br/>             | Describes how to use the Windows Touch API to build touch-enabled Windows-based applications.<br/>                                                                                                                                                                                                                                                                |
| [Keyboard and Mouse Input](https://msdn.microsoft.com/en-us/library/ms632585(v=VS.85).aspx)<br/>          | Describes how to capture user input from the keyboard and the mouse.<br/>                                                                                                                                                                                                                                                                                         |
| [Multipoint Mouse](https://msdn.microsoft.com/en-us/library/Ee906605(v=MSDN.10).aspx)<br/> | Describes the framework that enables you to build applications that support multiple mouse devices concurrently on one computer.<br/>                                                                                                                                                                                                                             |
| [Tablet PC](https://msdn.microsoft.com/en-us/library/ms704849(v=VS.85).aspx)<br/>          | Describes how to build applications for the Tablet PC, which is a fully functional personal computer geared for pen-enabled, handwriting-enabled, and speech-enabled applications. <br/>                                                                                                                                                                          |
| [DirectInput](https://msdn.microsoft.com/en-us/library/Ee416842(v=VS.85).aspx)<br/>      | Describes how to use the [DirectInput](https://msdn.microsoft.com/en-us/library/Ee416842(v=VS.85).aspx) API to process data from a joystick, or other game controller. DirectInput is legacy and not available for Windows Store apps. Use [XInput](https://msdn.microsoft.com/en-us/library/Hh405053(v=VS.85).aspx) instead.<br/>                                                                                      |
| [Text Services Framework](https://msdn.microsoft.com/en-us/library/ms629032(v=VS.85).aspx)<br/>   | Describes how to use the Microsoft Windows Text Services Framework (TSF), which is an API that enables text services to supply text to applications in a way that does not require any detailed knowledge of the applications that produce or receive the text. For example, a text service can provide text input from speech or handwriting input devices.<br/> |
