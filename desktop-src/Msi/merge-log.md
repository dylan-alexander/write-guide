# Merge.Log method

The **Log** method of the [**Merge**](merge-object.md) object writes a text string to the currently open log file.

## Syntax


```JScript
Merge.Log(
  Message
)
```



## Parameters

<dl> <dt>

*Message* 
</dt> <dd>

The text string to write to the log file.

</dd> </dl>

## Return value

This method does not return a value.

## C++

See [**Log**](https://msdn.microsoft.com/en-us/library/Aa369274(v=VS.85).aspx) function.
