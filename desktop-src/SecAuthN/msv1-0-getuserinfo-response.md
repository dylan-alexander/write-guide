# MSV1\_0\_GETUSERINFO\_RESPONSE

The MSV1\_0\_GETUSERINFO\_RESPONSE structure is obsolete.

For retrieving information about the workstation user, see the [**NetWkstaUserGetInfo**](https://msdn.microsoft.com/en-us/library/Aa370670(v=VS.85).aspx) function.
