# VSS Backups and Restores of the Cluster Database

Failover clustering uses the Volume Shadow Copy Service (VSS) for backing up and restoring the failover cluster configuration. For more information, see [Backing Up and Restoring the Failover Cluster Configuration Using VSS](https://msdn.microsoft.com/en-us/library/Dd264602(v=VS.85).aspx).
