# DWM Constants and Enumerations

This section contains information about the constants and enumerations used in the Desktop Window Manager (DWM) APIs.

## In this section



| Topic                                                                                     | Description                                                                                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM Blur Behind Constants**](dwm-bb-constants.md)<br/>                          | Flags used by the [**DWM\_BLURBEHIND**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_blurbehind) structure to indicate which of its members contain valid information.<br/>                                                                    |
| [**DWM Enable Composition Constants**](dwm-ec-constants.md)<br/>                   | Flags used by the [**DwmEnableComposition**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmenablecomposition)  function to change the state of DWM composition.<br/>                                                                             |
| [**DWM\_SOURCE\_FRAME\_SAMPLING**](/windows/desktop/api/Dwmapi/ne-dwmapi-dwm_source_frame_sampling)<br/>              | Flags used by the [**DwmSetPresentParameters**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetpresentparameters) function to specify the frame sampling type.<br/>                                                                            |
| [**DWM\_TAB\_WINDOW\_REQUIREMENTS**](/windows/desktop/api/dwmapi/ne-dwmapi-dwm_tab_window_requirements)<br/>          | Returned by DwmGetUnmetTabRequirements to indicate the requirements needed for a window to put tabs in the application title bar.<br/>                                                                    |
| [**DWM\_TNP Constants**](dwm-tnp-constants.md)<br/>                                | Flags used by the [**DWM\_THUMBNAIL\_PROPERTIES**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_thumbnail_properties) structure to indicate which of its members contain valid information.<br/>                                               |
| [**DWMFLIP3DWINDOWPOLICY**](/windows/desktop/api/Dwmapi/ne-dwmapi-dwmflip3dwindowpolicy)<br/>                         | Flags used by the [**DwmSetWindowAttribute**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetwindowattribute) function to specify the Flip3D window policy.<br/>                                                                               |
| [**DWMNCRENDERINGPOLICY**](/windows/desktop/api/Dwmapi/ne-dwmapi-dwmncrenderingpolicy)<br/>                           | Flags used by the [**DwmSetWindowAttribute**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetwindowattribute) function to specify the non-client area rendering policy.<br/>                                                                   |
| [**DWMTRANSITION\_OWNEDWINDOW\_TARGET**](/windows/desktop/api/Dwmapi/ne-dwmapi-dwmtransition_ownedwindow_target)<br/> | Identifies the target.<br/>                                                                                                                                                                               |
| [**DWMWINDOWATTRIBUTE**](/windows/desktop/api/Dwmapi/ne-dwmapi-dwmwindowattribute)<br/>                               | Flags used by the [**DwmGetWindowAttribute**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmgetwindowattribute) and [**DwmSetWindowAttribute**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetwindowattribute) functions to specify window attributes for non-client rendering.<br/> |
| [**GESTURE\_TYPE**](/windows/desktop/api/Dwmapi/ne-dwmapi-gesture_type)<br/>                                          | Identifies the gesture type specified in [**DwmRenderGesture**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmrendergesture).<br/>                                                                                                               |
