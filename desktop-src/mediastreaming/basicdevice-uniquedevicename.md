# BasicDevice.UniqueDeviceName property

Gets the unique device name (UDN) of the device.

This property is read-only.

## Syntax


```C++
HRESULT get_UniqueDeviceName(
  [out] HSTRING *value
);
```



## Property value

A pointer to the device’s model UDN.
