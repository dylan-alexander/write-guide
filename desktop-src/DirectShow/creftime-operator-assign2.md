# CRefTime.operator= method

The = operator assigns a new reference time.

## Syntax


```C++
CRefTime& operator=(
   const LONGLONG ll
);
```



## Parameters

<dl> <dt>

*ll* 
</dt> <dd>

New reference time, in 100-nanosecond units.

</dd> </dl>

## Return value

Returns a reference to the object.
