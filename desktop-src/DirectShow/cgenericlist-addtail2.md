# CGenericList.AddTail method

The `AddTail` method appends a list to the end of the list.

## Syntax


```C++
BOOL AddTail(
   CGenericList<OBJECT> pList
);
```



## Parameters

<dl> <dt>

*pList* 
</dt> <dd>

Pointer to the list to append.

</dd> </dl>

## Return value

Returns **TRUE** if successful, or **FALSE** otherwise.
