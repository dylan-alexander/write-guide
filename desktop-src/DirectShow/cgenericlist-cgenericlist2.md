# CGenericList.CGenericList constructor

Constructor method.

## Syntax


```C++
CGenericList(
   TCHAR *pName
);
```



## Parameters

<dl> <dt>

*pName* 
</dt> <dd>

</dd> </dl>

## Remarks

For efficiency, the `CGenericList` class maintains a cache of list nodes. This version of the constructor uses a default cache size.
