# WebWizardHost.Property property

Sets or retrieves a property's current value.

This property is read/write.

## Syntax


```JScript
Property = WebWizardHost.Property(bstrPropertyName)
WebWizardHost.Property = Property
```



## Property value

A variable of type [**BSTR**](https://msdn.microsoft.com/en-us/library/ms221069(v=VS.71).aspx) that specifies or receives the property.
