# SFVM\_SIZE message

Notifies the callback object that the folder view has been resized. Used by [**IShellFolderViewCB::MessageSFVCB**](https://msdn.microsoft.com/en-us/library/Bb774968(v=VS.85).aspx).


```C++
SFVM_SIZE
```



## Parameters

This message has no parameters.
