# CFolderItems class

**CFolderItems** is a collection of [**FolderItems**](folderitems.md). It implements the following interfaces: [**IPersistFolder**](/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ipersistfolder), [**IObjectWithSite**](https://msdn.microsoft.com/en-us/library/ms693765(v=VS.85).aspx), [**FolderItems3**](folderitems3-object.md), [**IObjectSafety**](https://msdn.microsoft.com/library/Aa768224(v=VS.85).aspx), [**IDispatch**](https://msdn.microsoft.com/en-us/library/ms221608(v=VS.71).aspx).

## Remarks

The **CFolderItems** class only enumerates items from a specific folder.
