# IDCompositionRectangleClip::SetTop methods

Animates or changes the value of the Top property of a clip rectangle. The Top property specifies the y-coordinate of the upper-left corner of the clip rectangle.

### Overload list



| Method                                                                                               | Description                                        |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| [**SetTop(float)**](https://msdn.microsoft.com/en-us/library/Hh448921(v=VS.85).aspx)                                     | Changes the value of the Top property.<br/>  |
| [**SetTop(IDCompositionAnimation\*)**](https://msdn.microsoft.com/en-us/library/Hh448922(v=VS.85).aspx) | Animates the value of the Top property.<br/> |
